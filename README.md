# SensorCPP
C++ Interface Libraries for common sensors, written for SBCs instead of microcontrollers
This is for the programmer who likes C++ and is tired of looking for python code and rewriting it.
When possible, manufacturer documentation is used to ensure compatibility and accuracy.

## System / SBC requirements
### g++
### cmake
### i2c-dev (apt) or libi2c-devel (yum)

# Devices Implemented

## Sensors
### I2C
AHT10 - Temperature and Humidity Sensor  

## SBCs Tested with this library
Raspberry Pi (using 3, no reason 4 and 5 wont work either)  
AMLogic LibreComputer LePotato
